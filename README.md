# zero-shot-transformer-app-review-analysis
A Zero-Shot Transformer Framework for Classifying Google Play Store Reviews  to Enhance App Usability

This repository presents a Zero-Shot Transformer-based framework for analyzing and classifying Google Play Store reviews to support data-driven usability evaluation. The system leverages advanced pre-trained language models—including BERT, RoBERTa, DeBERTa, and BART—to perform sentiment analysis and categorize user feedback into actionable issue types without requiring task-specific labeled data.

The framework implements a complete end-to-end pipeline, encompassing data acquisition, preprocessing, spam filtering, transformer-based sentiment ensemble modeling, and zero-shot classification. It further incorporates topic modeling, statistical validation, and visualization techniques to uncover patterns in user feedback and quantify relationships between sentiment, ratings, and usability concerns.

By combining zero-shot learning with explainable and scalable NLP techniques, this project demonstrates an efficient approach to extracting meaningful insights from large-scale user reviews. The methodology is designed to be reproducible and adaptable across applications, enabling developers and researchers to systematically identify usability issues and enhance application quality.
